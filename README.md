# Data_In_R: Statistical Analysis and Modeling in R

## Overview

This repository contains R code for performing various statistical analyses and modeling techniques on datasets. The code covers topics such as simple and multiple linear regression, statistical inferences and diagnostics, stepwise regression, model selection, advanced regression methods, and basic design and analysis of experiments.
